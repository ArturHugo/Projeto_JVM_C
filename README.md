# Leitor e Exibitor

Leitor e exibidor de arquivos no formato `.class` do Java.

## Compilação

```
gcc -Wall -Wextra -g -std=c99 -Iinclude src/*.c -o jvm
```

## Uso

```
./jvm <Arquivo.class>
```
